# Sales Data Analysis Using Excel
Project Overview
This project demonstrates a complete end-to-end data analysis workflow using Microsoft Excel, covering data cleaning, transformation, visualization, and summary reporting. The objective is to convert raw sales data into meaningful insights and visual reports using Excel’s analytical features.

Dataset Description
The dataset contains 200 sales records with the following attributes:
Order ID
Order Date
Customer Name
Region
Product
Category
Quantity
Unit Price
Discount
Sales Amount
Sales Representative

 Tools & Technologies
Microsoft Excel
Excel Functions & Formulas
Pivot Tables
Charts & Conditional Formatting

 Day-wise Workflow Summary
 Day 1 – Dataset Understanding
 Imported sales dataset into Excel
 Explored rows, columns, and data types
 Understood business meaning of each column

 Day 2 – Data Cleaning
 Removed blank and duplicate records
 Corrected data formats (dates, numbers)
 Standardized text values
 Ensured data consistency and accuracy

 Day 3 – Data Transformation & Analysis
 Created calculated columns (Total Sales, Profit, Sales Status)
 Applied Excel formulas: SUM, AVERAGE, COUNTIF, IF, MAX, MIN
 Performed sorting and filtering for focused analysis
 Used conditional formatting to highlight key values

 Day 4 – Data Visualization
 Created PivotTables to summarize data
 Designed charts:
 Column Chart (Sales by Region)
 Pie Chart (Category-wise Sales Contribution)
 Line Chart (Monthly Sales Trend)
 Added chart titles, labels, and formatting
 Analyzed trends, patterns, and performance insights

 Day 5 – Summary Analysis & Reporting
 Built a dedicated Summary sheet
 Displayed key metrics and KPIs
 Highlighted important findings and outliers
 Integrated charts for visual storytelling
 Finalized and organized the workbook for reporting

 Key Insights
 Electronics category contributes the highest revenue share
 South and East regions show strong sales performance
 Seasonal trends observed in monthly sales
 High-discount orders significantly impact revenue

 Project Files
 Sales_Data_Day3.xlsx – Cleaned and transformed data
 Sales_Data_Day4.xlsx – Charts and visual analysis
 Sales_Data_Final.xlsx – Final summary report

 Learning Outcomes
 Practical understanding of Excel-based data analysis
 Hands-on experience with formulas, PivotTables, and charts
 Ability to transform raw data into insights
 Improved reporting and presentation skills

 Conclusion
 This project highlights how Microsoft Excel can be effectively used as a powerful tool for data analysis and reporting. It demonstrates a structured approach from  raw data to actionable insights, making it suitable for academic projects, internships, and beginner data analyst roles.
